概要:
DXライブラリ(C++)を使った,縦スクロール,二人協力型弾幕シューティングゲームです.
情報科学研究部の1年生の有志グループでコンフェス2017用に制作したものを,
同意の上で課題として提出しています.

工夫した点:
・一番意識した元ネタは,上海アリス幻樂団様の"東方Project"シリーズで,
原作のように,主に三角比を用いて,美しい弾幕に見えるように工夫しました.
・一画面に弾が1000発も生成される場面と,逆に全く生成されない場面があるため,
弾を管理する配列を可変長配列(vector)にしました.
敵,自機が発射する武器も同じように可変長配列で実装しました.
・x,y座標を持つ移動するオブジェクトを、まとめて"Mover"という基底クラスから派生することで,
同じような処理を書かないようにしました.
・画像素材,音声素材は,可能な限り自分たちで自作しました.

操作方法(ゲームパッドに依存しないもの):
1P : WASD    ...移動	左シフト押しっぱなし...低速移動
2P : 矢印キー...移動	右シフト押しっぱなし...低速移動

引用について:
・SceneManager.cpp(SetSceneを除く)と,
BaseScene.hは,ほぼ館のコードをコピーしたものです.
そのほかにも,いろいろな実装で館を参考にしました.
・弾幕の形は東方projectの弾幕を移植したものが一部あります.
・DXライブラリの関数の使い方でライブラリ置き場,
渦巻き型弾幕の考え方に弾幕本を使用させていただきました.
(・その他c++の勉強やエラーの解決に様々なサイトを参考にしましたが,
どのサイトを参考にしたか記録していなかったです,申し訳ないです.)
